# Support

## Purdue AF mailing list

In case of problems, as well as for questions and suggestions,
please send a message to the following address:
[cms-af@groups.purdue.edu](mailto:cms-af@groups.purdue.edu).

This mailing list is used for important announcements and alerts about
new features.

## Subscribing to the mailing list

Users with Purdue email can subscribe to it by managing their
group membership at the [Microsoft Exchange admin center](https://admin.exchange.microsoft.com/#/):

1. Log in with Pudue career account.
2. Click "groups" in the sidebar.
3. Under the section "Groups I belong to", clink on the "Join..." icon.
4. Seacrh for `cms-af`.
5. Select "Purdue CMS Analysis Facility" and click "Join group".

Unsubscribing from the mailing list is managed via the same portal.
