# Interactive demos (notebooks)

These demos can be downloaded from the
[purdue-af-demos repository](https://github.com/PurdueAF/purdue-af-demos) and
executed as Jupyter Notebooks directly in your AF session.

* [ROOT C++ notebook example](root-cpp.md)
* [Accessing files in different storage locations](storage-data-access.md)
* [Plotting with `matplotlib`, `mplhep`, `pyROOT`](plotting.md)
* [Dask Gateway cluster setup](gateway-cluster.md)

To run them at Purdue AF, clone the repository into one of your
[storage volumes](../storage.md) and open the notebooks in JupyterLab:

```shell
git clone https://github.com/PurdueAF/purdue-af-demos.git
```
